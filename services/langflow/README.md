# Langflow Service

Langflow is a visual flow builder for LangChain applications.

## Access
- **URL**: http://localhost:7860
- **Container**: `nostack-langflow`

## Features
- Visual flow builder for LangChain
- Drag-and-drop interface
- Pre-built components
- Custom component support
- Flow persistence via PostgreSQL

## Data Storage
- **Flows**: Stored in PostgreSQL database `langflow`
- **Data**: Persisted in `data/langflow/`
