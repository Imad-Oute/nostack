# OpenWebUI Backend

This directory is populated automatically by the `setup.py` script.

It will clone the [OpenWebUI repository](https://github.com/open-webui/open-webui) here during the setup process.

If you want to manually set this up:
1.  Clear this directory (if not empty).
2.  Run: `git clone https://github.com/open-webui/open-webui.git .`
